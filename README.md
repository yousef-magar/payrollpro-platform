# PayrollPro

Modern Payroll Management System built with Vanilla JavaScript.

![PayrollPro Preview](./preview.png)

## 🚀 Features

- Excel & CSV Upload
- Smart Payroll Calculations
- Dynamic PDF Report Generation
- Authentication System
- IndexedDB Database
- Company Branding & Logo Support
- Customizable PDF Styling
- Responsive Dashboard UI
- Report History Management
- Drag & Drop Column Mapping
- Multi-Currency Support

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- IndexedDB
- jsPDF
- PapaParse
- SheetJS (XLSX)
- html2canvas
