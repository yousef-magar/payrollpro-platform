## ✨ Overview

**PayrollPro** is a fully client-side payroll management system designed to automate salary processing workflows for companies and HR teams.

Built entirely with **Pure JavaScript** — no frameworks, no backend, no external servers.

The system handles everything from:

* Employee payroll calculations
* Excel & CSV processing
* Dynamic payroll rules
* PDF report generation
* Authentication & session management
* IndexedDB local database storage
* Company branding customization

All running locally inside the browser.
